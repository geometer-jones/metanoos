# Architecture

This repository studies a complex-valued sequence architecture whose primitive
is not attention, but associative state composition.

Attention treats sequence positions as objects with intrinsic pairwise
similarities, then normalizes those similarities into weights. This architecture
treats sequence positions as generators of a composed relational state: a
monoid element that serves as a sufficient statistic from which later positions
extract readouts through normalized projection.

The prefix state is not an approximation to an attention matrix. It is a
different mathematical species.

## System Purpose

The system is a research testbed for a composed-state view of sequence modeling.

The core question is:

> Can a causal sequence be represented by an associative monoid element — a
> composed relational state whose content is complex, whose relevance mass is
> real, and whose readouts are normalized projections against the accumulated
> prefix?

The prefix state is an element of a monoid under chronological composition.
The identity element is `(1, 1, 0, 0)`. Associativity is the monoid law that
makes the prefix state well-defined independent of evaluation order, that
guarantees equivalence between full-sequence training and one-step inference,
and that makes the scan a genuine mathematical primitive rather than an
implementation convenience.

The phase split is the content of the primitive. The monoid law is its form.

## Ontology

The architecture does not begin with pairwise token comparison. It begins with a
generative procedure.

Each sequence position deposits a local relational contribution into a prefix
state. The prefix state is the observable memory of the sequence so far:

```text
P_t = A_1 o A_2 o ... o A_t
```

where each `A_t` is the contribution generated by position `t`, and `o` is an
associative chronological composition law.

The meaningful object is not a token in isolation and not a matrix of token-token
similarities. The meaningful object is the composed prefix state `P_t`, because
it is the minimal object from which future readouts can be extracted.

## State Space

Model states live in a complex vector space:

```text
z_t in C^d
```

Each position generates query, key, and value coordinates:

```text
q_t, k_t, v_t in C^d
```

The architecture separates two kinds of information:

- **relevance mass**, represented by nonnegative real gates;
- **relational content**, represented by complex amplitudes with phase.

For a complex vector `x`, the positive gate is:

```text
g(x) = elu(|x|) + 1
```

and the phase-bearing feature is:

```text
phi(x) = g(x) * x / max(|x|, eps)
```

Thus:

```text
q_gate_t = g(q_t)          in R_+^d
q_phase_t = phi(q_t)      in C^d
k_gate_t = g(k_t)         in R_+^d
k_phase_t = phi(k_t)      in C^d
```

The gates are phase-blind. The phase-bearing features retain rotational
structure.

## Local Generator

Each position contributes a local generator:

```text
S_t = v_t k_phase_t^*     in C^{d_v x d_k}
Z_t = k_gate_t            in R_+^{d_k}
```

`S_t` is complex relational content. It binds value amplitude to key phase.

`Z_t` is real relevance mass. It records how much positive key support has been
deposited.

The generator may also carry temporal transport factors:

```text
alpha_S,t in C
alpha_Z,t in R_+
```

`alpha_Z,t` attenuates real mass. `alpha_S,t` attenuates and rotates complex
content.

## Two-Channel Transport

The architecture separates information into two channels with independent
transport laws:

- **Rotational channel:** complex content `S` transported by complex
  `alpha_S`, which attenuates and rotates. Old content may arrive at the
  readout in a different phase relationship to newer content.
- **Real channel:** real support `Z` transported by real `alpha_Z`, which
  attenuates without rotation. Support only weakens; it does not curve.

This split is the architecture's distinctive structural commitment. It is
what separates the prefix state from a decay-weighted linear attention
accumulator: the rotational channel carries content that can interfere,
cancel, and accumulate phase curvature through time, while the real channel
preserves a strictly nonnegative notion of support against which those
complex readouts are normalized.

The empirical question is whether this rotational content helps or hurts for
actual sequence tasks.

## Associative Composition

A composed state is:

```text
A = (alpha_S, alpha_Z, S, Z)
```

where `S` is complex content and `Z` is real relevance mass.

Two chronological states compose as:

```text
A_left o A_right =
(
  alpha_S,right alpha_S,left,
  alpha_Z,right alpha_Z,left,
  alpha_S,right S_left + S_right,
  alpha_Z,right Z_left + Z_right
)
```

The operation is associative:

```text
(A o B) o C = A o (B o C)
```

This is the formal center of the architecture.

Associativity means the composed prefix is independent of evaluation grouping:

```text
((A_1 o A_2) o A_3) o A_4
=
A_1 o (A_2 o (A_3 o A_4))
```

The identity state is:

```text
I = (1, 1, 0, 0)
```

and the causal memory at time `t` is:

```text
P_t = A_1 o A_2 o ... o A_t
```

Because the same law defines both parallel prefix construction and recurrent
stepping, training and inference are two evaluation strategies for one
mathematical object.

## Measurement

After a prefix state has been composed, position `t` reads it through its query.

The complex numerator is:

```text
n_t = S_prefix,t q_phase_t
```

The real denominator is:

```text
d_t = <Z_prefix,t, q_gate_t> + eps
```

The measured output is:

```text
y_t = n_t / d_t
```

This is a normalized linear projection of the prefix state, not a Born-rule
measurement and not a weighted sum over an explicit set of pairwise
similarities. The Born-rule structure enters only at the vocabulary readout
(see Gauge Structure And Readout); the intermediate readout is a linear
projection of complex content divided by real support.

The denominator measures real support. The numerator measures complex relational
content. Their ratio extracts a normalized complex readout from the composed
state.

## Rotational Memory

Rotational memory is the empirical consequence of the two-channel transport
split. Whereas scalar decay only weakens the past, complex `alpha_S` makes
time a source of curvature: older evidence may arrive at the readout in a
different phase relationship to newer evidence, producing interference.

The open research questions:

- Does temporal rotation encode useful order structure?
- Does phase curvature help separate overlapping memories?
- Does destructive interference improve compression?
- Or does rotational memory destroy information that ordinary sequence tasks
  need preserved?

## Gauge Structure And Readout

The final vocabulary readout treats complex amplitudes as measurable quantities.

For vocabulary carrier `e_v`, the amplitude for token `v` is:

```text
a_v = <x_t, e_v>
```

The default measurement is the Born rule on the amplitude:

```text
logit_v = |a_v|^2
```

This readout is invariant under global U(1) phase rotation of `x_t`. The
observable output depends on relative phase structure, not absolute phase.

That invariance is not decorative. It defines a gauge freedom:

```text
x_t -> exp(i theta) x_t
```

under which Born-rule logits remain unchanged.

This invariance does not extend to the prefix state. The phase structure of
the composed state carries genuine rotational content that affects the
intermediate projection `y_t = S q_phase / (<Z, q_gate> + eps)`. Only the
final vocabulary readout is phase-invariant; the prefix state itself is not
gauge-equivalent to a phase-rotated copy, and the rotational channel is not
gauge-redundant.

An alternate real readout,

```text
logit_v = Re(a_v)
```

breaks that invariance and exposes absolute phase sensitivity. It is therefore a
useful comparison point, but it is not the natural measurement for the
relational-state ontology.

## Block Dynamics

A model block is a residual transformation of complex state.

Conceptually:

```text
x <- x + Mix(Norm(x))
x <- x + MLP(Norm(x))
```

The normalization rescales magnitude while preserving phase.

The pointwise nonlinearity is phase-preserving:

```text
modReLU(z) = relu(|z| + b) * z / (|z| + eps)
```

The block therefore alternates between:

- composing relational sequence memory;
- applying local complex nonlinear transformation;
- preserving residual access to prior complex state.

## Contrast With Attention

Attention constructs an explicit relation between positions:

```text
score(t, s) = similarity(q_t, k_s)
weight(t, s) = normalize_s(score(t, s))
output_t = sum_s weight(t, s) v_s
```

Its ontology is pairwise: sequence positions are compared directly, and the
resulting normalized matrix mediates retrieval.

The composed-state architecture instead constructs a prefix sufficient statistic:

```text
A_s -> P_t = A_1 o ... o A_t
y_t = measure(P_t, q_t)
```

Its ontology is generative and relational: positions deposit contributions into
a causal state, and queries measure that state.

Attention can be used as a baseline, but it should not be treated as the parent
object. The prefix state is not a low-rank attention matrix, a softmax
approximation, or a computational trick for attention. It is the object being
studied.

## Mathematical Flow

At the sequence level:

```text
tokens
  -> complex token states
  -> local relational generators
  -> associative prefix states
  -> query-conditioned measurements
  -> complex hidden states
  -> gauge-aware vocabulary measurement
  -> real logits
```

Within the composed-state primitive:

```text
z_t
  -> q_t, k_t, v_t
  -> q_gate_t, q_phase_t, k_gate_t, k_phase_t
  -> A_t = (alpha_S,t, alpha_Z,t, S_t, Z_t)
  -> P_t = A_1 o ... o A_t
  -> y_t = measure(P_t, q_t)
```

## Core Invariants

The architecture is organized around these invariants:

- **Associativity:** composition is independent of grouping.
- **Causality:** position `t` measures only the prefix state through `t`.
- **Training/inference equivalence:** scan and recurrence are evaluations of
  the same law.
- **Real relevance:** support and normalization are nonnegative real quantities.
- **Complex content:** relational memory retains phase-bearing amplitudes.
- **Rotational transport:** old content may rotate as well as attenuate.
- **Interference:** equal support can still produce cancellation or
  reinforcement.
- **Gauge freedom:** Born-style output depends on relative, not absolute, phase.

## Research Boundary

The repository is a controlled testbed for the composed-state hypothesis.

The important boundary is between:

- the **primitive**, an associative complex relational state with real support;
- the **experiments**, which ask whether that primitive produces useful sequence
  behavior.

New work should preserve this boundary. Changes to the primitive should clarify
or test the composition law, the transport factors, the measurement, or the
gauge structure. Changes to tasks and baselines should remain evidence about the
primitive, not replacements for the primitive itself.
